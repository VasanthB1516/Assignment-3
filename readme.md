This project is a fully static front-end replica of the NoBroker homepage built using HTML5 and CSS3.


Key Features Implemented
	•	Fixed top navigation bar with aligned menu items
	•	Hero section with search functionality
	•	Tab-based property categories (Buy / Rent / Commercial)
	•	Loan eligibility banner
	•	Services section with icon layout
	•	“Why Use NoBroker” feature highlight section
	•	Business assist section for builders
	•	Statistical difference section
	•	Customer testimonial cards
	•	Embedded YouTube testimonial video
	•	App download promotional section
	•	Large multi-column property links section
	•	Services tag-based grid layout
	•	Fully structured footer with social icons and store badges

Technical Highlights
	•	Semantic HTML5 structure
	•	Flexbox-based layout system
	•	Responsive design using wrapping and flexible widths
	•	Clean alignment and spacing control
	•	Proper content grouping with reusable section structures
	•	Embedded external media (YouTube iframe)
	•	Visual consistency across multiple sections

Learning Outcomes
	•	Handling large multi-section layouts
	•	Managing spacing and alignment issues
	•	Working with flexbox deeply
	•	Structuring complex footer grids
	•	Maintaining consistent UI hierarchy
	•	Debugging layout gaps and spacing conflicts
